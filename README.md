#Overview :-
This is a full-stack Job Portal application built to connect job seekers and recruiters. Users can browse jobs, apply, and manage applications, while recruiters can post and manage job listings.

#Tech Stack
1. Frontend
  a. React.js
  b. Tailwind CSS
  c. Axios
  d. React Router
2. Backend
  a. Node.js
  b. Express.js
  c. MongoDB
  d. JWT Authentication

#Features
1. User (Job Seeker)
  a. Register / Login
  b. Browse all jobs
  c. Apply to jobs
  d. View applied jobs
  e. Update profile
2. Recruiter / Admin
  a. Create job postings
  b. Edit / Delete jobs
  c. View applicants
  d. Manage job listings
